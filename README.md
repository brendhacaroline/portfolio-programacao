## Linguagens utilizadas
- C

## Projetos em C

### VectorStats
Programa em C com menu interativo que permite:
- inserir valores em um vetor
- mostrar os valores
- calcular soma, média, mínimo e máximo
- alterar valores do vetor

### Number Guessing Game
Jogo em C onde o utilizador tenta adivinhar um número gerado aleatoriamente pelo programa.

Funcionalidades:
- menu interativo
- definição do limite do número
- definição do número de tentativas
- sistema de ajuda

### Calculator Menu
Calculadora simples em C com menu interativo utilizando `switch`.

Operações disponíveis:
- soma
- subtração
- multiplicação
- divisão (com verificação para evitar divisão por zero)

## Projetos em Java

### MathMenuApp
Aplicação em Java com menu interativo que permite realizar várias operações matemáticas.

Operações disponíveis:
- soma
- subtração
- multiplicação
- divisão
- verificação de número par/ímpar
- equações e cálculos simples.

Tecnologias utilizadas:
- Java
- Scanner para entrada de dados
- Métodos para organização do código
- Estruturas condicionais e Repetição
- Menu interativo com switch

### Car Price Calculator

Aplicação em Java que calcula o preço final de um automóvel com base no preço de fábrica, impostos e percentagem do revendedor.

Funcionalidades:
- seleção da marca do automóvel
- cálculo automático do imposto (45%)
- cálculo da percentagem do revendedor
- cálculo do preço final

Tecnologias utilizadas:
- Java
- Scanner para entrada de dados
- switch para menu interativo
